### 3.1 Requesitos Funcionais

RF-010 – Gerenciamento de Janela de Reserva

Versão Atual: 2.0.0

Descrição: O sistema deve validar a data da reserva em relação à data atual. O período padrão permitido é de 24 horas a 6 meses de antecedência.

Regras de Exceção:

Clientes VIP: Podem estender a janela de reserva para até 1 ano.

Última Hora (Last Minute): Reservas com menos de 2 horas de antecedência são permitidas, desde que haja disponibilidade imediata.

Regra de Preço: Incidência automática de taxa adicional de 15% sobre o valor base para reservas efetuadas na janela de "Última Hora".
